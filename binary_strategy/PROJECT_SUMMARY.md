## 核心创新点

### 1. 动态重连逻辑 ⭐⭐⭐⭐⭐
**这是项目最大的创新和技术亮点**

```cpp
// 当单位死亡时，自动将子节点挂载到最近存活的祖先节点
void reconnectChildren() {
    StrategyNode* newParent = findNearestAliveAncestor();
    
    if (leftChild && leftChild->isActive) {
        if (newParent) {
            // 尝试挂载到新父节点
            if (!newParent->leftChild) {
                newParent->leftChild = leftChild;
                leftChild->parent = newParent;
                updateDepth(leftChild, newParent->depth + 1);
            } else if (!newParent->rightChild) {
                newParent->rightChild = leftChild;
                leftChild->parent = newParent;
                updateDepth(leftChild, newParent->depth + 1);
            }
        }
        leftChild->handleRootStatus();
    }
}
```

**意义:**
- 真正实现了动态二叉树维护
- 防止树断裂，保持阵型完整性
- 实战应用二叉树数据结构

### 2. Root加成衰减系统 ⭐⭐⭐⭐
```cpp
// Root加成随距离衰减
void applyRootBonus(int rootDepth) {
    depth = rootDepth;
    if (isRoot) {
        // Root单位获得特殊加成
        Buff rootBuff(BuffType::ROOT_BONUS, 30, -1, true);
        activeBuffs.push_back(rootBuff);
    } else {
        // 非Root单位的加成随深度衰减
        int distanceFromRoot = rootDepth - depth;
        if (distanceFromRoot > 0) {
            int bonusPercent = max(5, 30 - distanceFromRoot * 5);
            Buff distanceBuff(BuffType::ROOT_BONUS, bonusPercent, -1, true);
            activeBuffs.push_back(distanceBuff);
        }
    }
}
```

**意义:**
- 体现树结构的重要性
- 鼓励合理布局阵型
- 教学价值：理解树深度概念

### 3. Roguelike奖励系统 ⭐⭐⭐
```cpp
// 生成3个随机奖励选项（单位或Buff）
void generateRewards() {
    currentRewards.clear();
    
    for (int i = 0; i < 3; ++i) {
        bool isUnit = (rand() % 100) < 60; // 60%概率是新单位
        
        if (isUnit) {
            currentRewards.push_back(RoguelikeReward(getRandomUnitType()));
        } else {
            // Buff奖励
            BuffType buffType = getRandomBuffType();
            int bonusValue = 10 + (rand() % 20);
            currentRewards.push_back(RoguelikeReward(buffType, bonusValue));
        }
    }
}
```

**意义:**
- 增加游戏可玩性
- 训练决策能力
- 理解随机性和策略性
